Markdown PDF Editor Pro (V4.1)
박중희 (연세대학교 심리학과 겸임교수)
연구방법론, 인공지능, 인지과학, 인지공학 심리학 

강력한 실시간 A4 미리보기와 정밀한 PDF 내보내기 기능을 갖춘 웹 기반 마크다운 에디터

본 프로젝트는 단순한 마크다운 뷰어를 넘어, 실제 인쇄 및 PDF 저장 시의 레이아웃을 실시간으로 확인하며 문서를 작성할 수 있도록 설계된 전문 편집 도구입니다. 특히 외부 이미지 서버의 보안 정책(CORS) 문제를 해결하기 위한 로컬 업로드 기능을 포함하고 있습니다.

✨ 핵심 기능 (Key Features)

1. 🖼️ 완벽한 이미지 관리 (CORS 문제 해결)

IMG URL: 온라인 이미지 주소를 직접 삽입할 수 있습니다.

IMG UPLOAD: 내 컴퓨터의 이미지 파일을 직접 업로드하여 Base64 데이터로 삽입합니다. 이 방식은 외부 서버 보안 정책에 영향을 받지 않아 PDF 저장 시 이미지가 누락되지 않는 100% 안정성을 보장합니다.

2. 📊 스마트 표(Table) 편집기

표 자동 생성: 버튼 클릭 한 번으로 마크다운 표준 표 템플릿을 삽입합니다.

정밀 조작: 커서가 위치한 행 아래에 빈 행을 추가(Row+)하거나, 표 전체의 오른쪽에 열을 확장(Col+)하는 스마트 로직이 탑재되어 있습니다.

행 복제: 표 안에서 단축키 하나로 현재 행을 즉시 아래로 복사할 수 있습니다.

3. 📄 실시간 PDF/A4 미리보기

Page Break 가이드: <div class="page-break"></div> 태그 사용 시, 미리보기 화면에 파란색 점선 가이드가 나타나 실제 인쇄 범위를 직관적으로 확인할 수 있습니다.

A4 레이아웃: 실시간 미리보기 창은 실제 A4 규격(210mm)을 유지하여 정확한 레이아웃 설계가 가능합니다.

4. ⌨️ 생산성 도구

스크롤 동기화 (Scroll Sync): 편집창과 미리보기 창의 스크롤이 유기적으로 연동됩니다.

강력한 단축키:

Ctrl + B / I: 굵게 / 기울임

Ctrl + .: 인용구 (>)

Ctrl + Z: 실행 취소 (Undo)

Ctrl + Enter: 페이지 나누기 지점 삽입

Ctrl + Shift + Enter: 강제 줄바꿈 (</br>)

Shift + Alt + ↓: 현재 행 아래로 복제 (표 전용)

5. 🧪 학술 및 기술 문서 지원

수식 지원 (LaTeX): KaTeX 라이브러리를 통해 인라인($) 및 블록($$) 수식을 미려하게 렌더링합니다.

코드 하이라이팅: Python, JS 등 다양한 언어의 코드 블록을 지원합니다.

6. 💾 멀티 포맷 저장

PDF 저장: 인쇄 최적화된 PDF 생성.

MD 저장: 마크다운 원문 보존.

TXT 저장: 순수 텍스트 추출.

🚀 시작하기 (Getting Started)

본 프로젝트는 별도의 서버 설치가 필요 없는 Single-file HTML 솔루션입니다.

index.html 파일을 다운로드합니다.

최신 브라우저(Chrome, Edge 권장)에서 파일을 엽니다.

바로 편집을 시작하거나 마크다운 파일을 업로드하여 사용하세요.

🛠️ 사용된 기술 (Tech Stack)

UI/UX: Tailwind CSS

Markdown Engine: Marked.js

PDF Generation: html2pdf.js

Mathematics: KaTeX

Syntax Highlighting: highlight.js
