# Markdown PDF Editor Pro (V4.1)

강력한 실시간 A4 미리보기와 정밀한 PDF 내보내기 기능을 갖춘 웹 기반 마크다운 에디터
html live](https://shoutjoy.github.io/HTMLlive/)


**제작자: 박중희 (연세대학교 심리학과 겸임교수)
**

본 프로젝트는 단순한 마크다운 뷰어를 넘어, 실제 인쇄 및 PDF 저장 시의 레이아웃을 실시간으로 확인하며 문서를 작성할 수 있도록 설계된 전문 편집 도구입니다.

✨ 주요 기능 상세 설명 및 예제 (Features & Examples)


**1. 🖼️ 완벽한 이미지 관리 (CORS 문제 해결)**

웹상의 이미지를 가져오거나 내 컴퓨터의 파일을 직접 문서에 포함할 수 있습니다.

IMG URL (웹 주소 방식): 인터넷에 공개된 이미지 경로를 사용합니다.

예제: ![로고](https://example.com/logo.png)

설명: 간단하지만, 해당 서버가 보안(CORS) 정책을 사용하면 PDF 저장 시 이미지가 누락될 수 있습니다.

IMG UPLOAD (로컬 업로드 방식) - 권장:

방법: 툴바의 IMG UPLOAD 클릭 후 파일 선택.

결과: <img src="data:image/png;base64,..." style="width:300px;"> 형태로 삽입됩니다.

설명: 이미지가 텍스트 데이터로 문서에 직접 포함되므로, 오프라인 상태에서도 보이며 PDF 저장 시 100% 출력됩니다.

2. 📊 스마트 표(Table) 편집기

마크다운 표 작성이 서툴러도 버튼과 단축키만으로 정교한 표를 만들 수 있습니다.

표 생성 예제:

| 항목 | 값 | 비고 |
| :--- | :---: | ---: |
| 데이터1 | 100 | 완료 |


Row+ (행 추가): 표의 특정 칸에 커서를 두고 클릭하면, 아래에 똑같은 칸수의 빈 줄이 생깁니다.

Col+ (열 추가): 클릭 시 표 전체의 오른쪽에 새로운 열 한 줄이 일괄 추가됩니다.

행 복제 (Shift + Alt + ↓): 현재 타이핑 중인 줄의 내용을 그대로 아래 줄에 복사합니다. 반복적인 데이터 입력 시 매우 유용합니다.

3. 📄 실시간 PDF/A4 미리보기 및 페이지 제어

이 에디터의 핵심은 **"보이는 대로 인쇄된다"**는 점입니다.

페이지 나누기 (Page Break):

입력: <div class="page-break"></div> 또는 단축키 Ctrl + Enter.

효과: 미리보기 창에 파란색 점선 가이드가 나타나며, PDF 저장 시 해당 지점부터 무조건 다음 페이지에 인쇄됩니다.

A4 고정 레이아웃: 미리보기 창의 폭이 210mm(A4 표준)로 고정되어 있어, 표가 옆으로 잘리는지 사전에 확인할 수 있습니다.

4. ⌨️ 생산성 단축키 시스템

- **Ctrl + B / I**: 굵게 / 기울임
- **Ctrl + .**: 인용구 (>)
- **Ctrl + Enter**: 페이지 나누기 지점 삽입
- **Ctrl + Shift + Enter**: 줄바꿈 삽입 (</br>)
- **Ctrl + Z**: 실행 취소 (Undo)
- **Shift + Alt + ↓**: 표 안에서 현재 행 복제

5. 🧪 학술 및 기술 문서 지원 (LaTeX & Code)

수식과 프로그래밍 코드를 깔끔하게 정리할 수 있습니다.

수식 (LaTeX):

인라인: $a^2 + b^2 = c^2$ (문장 안에 삽입)

블록: $$E = mc^2$$ (중앙 정렬된 독립된 수식)

코드 블록:

사용법: Code 버튼 클릭 시 아래와 같은 블록이 생성됩니다.

def calculate_statistics(data):
    # 심리통계 분석 코드 예시
    return sum(data) / len(data)


6. 💾 멀티 포맷 저장 및 관리

PDF 저장: 실제 A4 인쇄물과 동일한 결과물 생성.

MD 저장: 나중에 다시 편집할 수 있도록 마크다운 원본 파일(document.md)로 저장.

TXT 저장: 서식 없이 내용만 빠르게 텍스트 파일(document.txt)로 저장.
